# Spectra Media — Projections vidéo pour danse & concerts

Landing page statique pour présenter les services de projections vidéo et de scénographie
de Vincent (Spectra Media) auprès des écoles de danse, compagnies, groupes et salles.

## Contenu

- `index.html` — page principale (SEO optimisé, intégration d'une vidéo YouTube d'exemple)
- `static/css/styles.css` — styles
- `static/img/hero-projection.svg` — visuel de couverture
- `static/img/favicon-spectra.svg` — favicon minimal
- `robots.txt`
- `sitemap.xml`

Remplace le domaine `https://video-for-dancers-and-musicians.example` par ton vrai domaine
dans `index.html`, `robots.txt` et `sitemap.xml` une fois déployé.

## Déploiement

Tu peux :

- pousser ces fichiers dans ton dépôt GitHub `video_for_dancers_and_musicians`
- connecter le repo à Vercel / Render / Netlify pour un déploiement en quelques clics.
